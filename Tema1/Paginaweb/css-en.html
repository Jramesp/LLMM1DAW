Property 1: color

Defines the color of the text.

p {
    color: blue;
}

Result:

This text is blue.

More information on W3C

Property 2: font-size

Defines the font size.

h1 {
    font-size: 24px;
}

Result:

This text has a font size of 24px.

More information on W3C

Property 3: margin

Defines the outer space of an element.

div {
    margin: 20px;
    background-color: #f0f0f0;
}

Result:

This block has a margin of 20px around it.

More information on W3C

Property 4: padding

Defines the inner space of an element.

div {
    padding: 20px;
    background-color: #e0e0e0;
}

Result:

This block has an inner padding of 20px.

More information on W3C

Property 5: background-color

Defines the background color of an element.

div {
    background-color: lightblue;
}

Result:

This block has a light blue background color.

More information on W3C
