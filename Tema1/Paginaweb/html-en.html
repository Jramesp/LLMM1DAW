Tag 2: <ul>

It is used to create unordered lists.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Result:

  • Item 1
  • Item 2
  • Item 3

More information on W3C

Tag 3: <table>

It is used to create tables.

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Juan</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Maria</td>
        <td>30</td>
    </tr>
</table>

Result:

Name Age
Juan 25
Maria 30

More information on W3C

Tag 4: <a>

It is used to create links to other pages or resources.

<a href="https://example.com" target="_blank">Visit Example</a>
