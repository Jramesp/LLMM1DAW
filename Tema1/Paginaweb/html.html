Etiqueta 2: <ul>

Se utiliza para crear listas no ordenadas.

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

Resultado:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Más información en W3C

Etiqueta 3: <table>

Se utiliza para crear tablas.

<table border="1">
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Juan</td>
        <td>25</td>
    </tr>
    <tr>
        <td>María</td>
        <td>30</td>
    </tr>
</table>

Resultado:

Nombre Edad
Juan 25
María 30
