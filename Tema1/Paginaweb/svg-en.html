Tag 1: <circle>

It is used to draw circles in an SVG graphic.

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" fill="blue" />
</svg>

Tag 2: <rect>

It is used to draw rectangles or squares.

<svg width="200" height="100">
    <rect x="10" y="10" width="180" height="80" fill="red" stroke="black" stroke-width="2" />
</svg>

Result:

Tag 3: <line>

It is used to draw lines.

<svg width="200" height="100">
    <line x1="10" y1="10" x2="190" y2="90" stroke="blue" stroke-width="2" />
</svg>

Tag 4: <polygon>

It is used to draw polygons by defining multiple points.

<svg width="200" height="100">
    <polygon points="50,10 150,10 100,90" fill="green" stroke="black" stroke-width="2" />
</svg>

Tag 5: <text>

It is used to add text within SVG graphics.
